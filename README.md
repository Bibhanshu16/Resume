## Features
- Clean and simple resume layout  
- Fully responsive design  
- Clickable certification links  
- Easy to update and maintain  
- Hosted and auto-deployed using Vercel  
- Sections included:
  - Summary  
  - Education  
  - Technical Skills  
  - Certifications  
  - Awards & Activities  
  - Projects
